# Icarus
Icarus est un projet créé durant le Web Basement (HTML) 2026 de la Haute-École Albert Jacquard. Le Web Basement est une jam organisée par l'option Interface Design Development de la HEAJ. Durant cette jam, nous avions une journée pour raconter adapter un conte dans un thème futuriste. 
## Les membres
Ce projet a été créé par trois personnes avec chacun un rôle distinct afin d'être sur tous les fronts durant la journée :
- Arthur Halbart : direction artistique et design
- Lucas Labiouse : design et maquettage sur Figma
- Gabriel Manciu : intégration de la maquette
